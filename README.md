# Hearthside Grove Motorcoach Resort Dashboard

Static executive dashboard demo for preview and deployment.

## Local preview

Run a simple static server from this folder:

```sh
python3 -m http.server 3005
```

Then open:

```sh
http://localhost:3005
```

## Deploy

Recommended: connect this repo to Vercel.

Because this is a static app, no build command is required.

## Files

- `index.html`
- `src/styles.css`
- `src/main.js`
- `src/mockData.js`
